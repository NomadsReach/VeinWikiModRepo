Step 5: Packaging Your Mod

1. Create Mod Folder Structure

  1. Go to where you've been saving your modified .uasset files.
  2. Delete the .json and .bak files.
  3. Create a folder for your mod. The name must end with _P.
    • Example: LxEZLockpicking_P
  4. Inside that folder, create the following subdirectory structure: Vein/Content/Vein/LockTypes/

    5. The path must exactly match the asset path shown in FModel (e.g., "Vein" appears twice).

  5. Copy all of your modified lock files (.uasset and .uexp) into the final LockTypes folder.

2. Package with UnrealPak

  1. Move your entire mod folder (the one ending in _P) to the directory where your UnrealPak.exe is located.
  2. Drag your mod folder onto the UnrealPak-With-Compression.bat script.
  3. A command prompt will open and show the packing progress. Press any key to close it when it's finished.
  4. A new .pak file will be created in your UnrealPak directory (e.g., LxEZLockpicking_P.pak).

3. Install and Test

  1. Move your new .pak file to the game's Paks folder:
    C:\Program Files (x86)\Steam\steamapps\common\Vein\Vein\Content\Paks
  2. Load the game and test your mod!

Testing Note: There's a quirk with lockpicking where setting the multiplier to 10 makes it so you unlock by moving the pick ALL THE WAY to the right and pressing D, rather than being able to do it anywhere.
