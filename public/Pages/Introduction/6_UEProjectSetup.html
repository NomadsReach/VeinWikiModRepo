Step 6: Unreal Engine Project Setup

For more advanced mods, such as creating new assets or logic, you will need to use Unreal Engine itself. This section covers the initial project setup.

1. Install Unreal Engine and Prerequisites

Start by installing Unreal Engine 5.4 from the Epic Games Launcher.

Important - Windows SDK Required: Before using Unreal Engine for modding, ensure you have the Windows SDK installed. Without it, UE will display errors when packaging or cooking content.

Download the Windows SDK from: Microsoft Windows SDK

Note: If you install the SDK while Unreal Engine is open, you must close and reopen the editor for the changes to take effect.

2. Creating a VEIN Project

  1. Launch Unreal Engine 5.4
  2. Select the Games category on the left, then choose the Blank template.
  3. Set your project location (choose somewhere with plenty of disk space).
  4. Set the Project Name to exactly Vein.
  5. Click Create.

3. Advanced Project Configuration (for Logic Mods)

The following settings are required for creating Blueprint and Logic mods that will be loaded with UE4SS.

  1. In your new project, go to Edit → Project Settings.
  2. Set the "Project Name" field (likely blank) to Vein.
  3. In the left panel, find the Packaging section.
  4. Find and disable "Use Io Store".
  5. Find and enable "Generate Chunks".
  6. Next, go to Edit → Editor Preferences.
  7. In the left panel, find the Experimental section.
  8. Under the "User Interface" dropdown, enable "Allow ChunkID Assignments".

Your project is now configured for both basic content cooking and advanced logic modding.
