## Modding Guide

### Creating a Custom Weapon

1. Choose a base weapon type (ranged or melee)
2. Copy existing weapon JSON
3. Modify key properties:
   - Damage values
   - Range
   - Accuracy
   - Fire rate (ranged)
   - Swing speed (melee)
   - Durability
   - Ammunition type (ranged)
4. Update asset references (model, sounds, effects)
5. Test balance and feel

### Key Weapon Properties

- **BaseDamage**: Base damage per hit
- **DamageType**: Type of damage dealt
- **Range**: Effective range
- **Accuracy**: Hit chance modifier
- **FireRate**: Shots per minute (ranged)
- **MagazineSize**: Ammo capacity (ranged)
- **ReloadTime**: Time to reload (ranged)
- **Durability**: Weapon condition degradation
- **WeaponWeight**: Affects stamina usage

### Creating a Custom Damage Type

1. Copy existing damage type
2. Define damage modifiers
3. Set effects on hit (bleeding, burning, etc.)
4. Configure resistance types
5. Link to condition effects

### Balancing Tips

- Start with vanilla weapon as baseline
- Adjust one property at a time
- Test against various enemy types
- Consider ammo availability for ranged weapons
- Balance melee weapons with risk/reward
