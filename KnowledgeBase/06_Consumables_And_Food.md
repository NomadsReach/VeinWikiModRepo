## Food System

Food in VEIN has multiple properties:
- Hunger restoration
- Thirst impact
- Health effects
- Calorie value
- Freshness/spoilage
- Cooking requirements

## Cooking System

Raw food can be cooked to:
- Improve nutritional value
- Remove negative effects
- Extend shelf life
- Improve taste

## Modding Guide

### Creating Custom Food Items

1. Copy existing food item
2. Set nutritional properties:
   - Hunger restoration
   - Thirst impact
   - Calories
   - Health effects
3. Configure spoilage:
   - Fresh duration
   - Spoiled effects
4. Set cooking parameters (if applicable)

### Creating Cooked Dishes

1. Create recipe in recipe system
2. Define ingredient requirements
3. Set cooking time and workbench
4. Configure final dish properties
5. Add to cooking skill progression

### Balancing Food

- Common foods should be less nutritious
- Cooked food should be more effective
- Rare ingredients should create powerful dishes
- Spoilage should encourage planning
- Water should be consistently important
