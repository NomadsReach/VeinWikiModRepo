## Modding Guide

### Creating Custom Buildings

1. Choose base building template
2. Modify mesh references
3. Set collision properties
4. Configure door placements
5. Add interior furniture spawns
6. Set loot spawn points

### Working with Doors

Doors in VEIN can be:
- Locked (requiring keys or lockpicking)
- Barricaded (blocking zombie entry)
- Destructible (can be broken down)
- Interactable (player can open/close)

### Furniture Placement

Furniture can serve as:
- Storage containers
- Crafting stations
- Rest points
- Decorative elements
- Cover in combat

### Building Components

- Walls and foundations
- Roofs and ceilings
- Windows and doors
- Stairs and elevators
- Electrical fixtures
- Plumbing fixtures
