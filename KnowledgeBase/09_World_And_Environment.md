## Modding Guide

### World Building

Environment modding involves:
- Terrain modification
- Foliage placement
- Weather configuration
- Day/night cycle
- Seasonal changes

### Creating Custom Weather

1. Define weather effects:
   - Visual effects (rain, snow, fog)
   - Audio cues
   - Gameplay impacts
2. Set transition conditions
3. Configure duration ranges

- Use foliage for visual variety
- Consider performance impact
- Match biome themes
- Allow for destruction/harvesting
