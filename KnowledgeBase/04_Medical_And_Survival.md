## Modding Guide

### Creating Medical Items

1. Copy existing medical item template
2. Define healing/effect properties:
   - Health restoration
   - Condition removal
   - Buff duration
   - Side effects
3. Set usage parameters:
   - Use time
   - Cooldown
   - Stack effects
4. Configure addiction risk (if applicable)

### Creating Custom Conditions

1. Define condition effects:
   - Stat modifiers
   - Visual effects
   - Duration
   - Progression stages
2. Set acquisition methods
3. Define cure conditions
4. Link to medical items

### Creating Custom Stats

1. Define stat parameters:
   - Min/max values
   - Depletion rate
   - Regeneration rate
2. Link to gameplay effects
3. Set UI display properties
4. Configure death threshold (if applicable)

### Balancing Survival Systems

- Medical items should be scarce but effective
- Conditions should have clear symptoms and cures
- Stats should deplete at realistic rates
- Addictions should present meaningful choices
- Death from conditions should be preventable with preparation
