# Chapter 2: Recipes and Crafting

## Overview

The VEIN crafting system allows players to create items from raw materials using recipes at various workbenches. The system consists of three main components:

- **Recipes (CR_/RP_)**: Define what can be crafted (207 recipes)
- **Workbench Types (WT_)**: Define where items can be crafted (83 types)
- **Recipe Types (RT_)**: Categorize recipes by skill (23 types)

## Recipe Structure

Recipes typically contain:

- **Inputs**: Required ingredients and quantities
- **Outputs**: Resulting items and quantities
- **WorkbenchType**: Where it can be crafted
- **RecipeType**: Category/skill requirement
- **CraftTime**: Time to complete crafting
- **SkillRequirement**: Required skill level
- **UnlockCondition**: How recipe is learned

## Modding Guide

### Creating a New Recipe

1. Copy an existing recipe JSON file
2. Rename the asset appropriately (CR_YourRecipeName)
3. Update the NameMap to include all referenced assets
4. Modify recipe properties:
   - Input items and quantities
   - Output items and quantities
   - Workbench requirement
   - Craft time
   - Skill requirements
5. Add recipe to appropriate recipe type
6. Convert back to .uasset
7. Pack into mod

### Example Recipe Structure

A typical recipe references:
- Input items (array of item blueprints + quantities)
- Output items (array of item blueprints + quantities)
- WorkbenchType reference
- RecipeType reference
- Sound cues for crafting
- Skill stat references

### Testing Your Recipe

1. Ensure all referenced items exist
2. Test at appropriate workbench
3. Verify craft time is reasonable
4. Check output quantities are correct
5. Test skill progression if applicable
