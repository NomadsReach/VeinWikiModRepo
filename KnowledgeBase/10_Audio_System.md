## Audio Types

- Sound Cues (SC_): Gameplay sound effects
- Music: Background music tracks
- Ambience: Environmental sounds
- Dialogue: Character voices
- UI Sounds: Interface feedback

## Modding Guide

### Adding Custom Sounds

1. Prepare audio file (WAV format recommended)
2. Import to Unreal Engine
3. Create Sound Cue asset
4. Configure attenuation and properties
5. Reference in blueprints

### Audio Best Practices

- Use appropriate bit rates
- Configure 3D spatial audio
- Set correct attenuation
- Test volume levels
- Consider performance impact
