## Hunting System

Wildlife can be hunted for:
- Meat
- Hides
- Bones
- Other materials

## Modding Guide

### Creating Custom Animals

1. Define animal behavior:
   - Movement patterns
   - Aggression level
   - Flee conditions
2. Set loot drops
3. Configure spawn conditions
4. Set health and damage values

### Creating Zombie Variants

1. Copy base zombie template
2. Modify appearance
3. Adjust stats (health, damage, speed)
4. Set special abilities
5. Configure spawn conditions
6. Define loot drops

### Balancing Creatures

- Passive animals should flee from player
- Aggressive animals should have clear audio/visual warnings
- Zombie variants should scale with player progression
- Loot should justify hunting effort
