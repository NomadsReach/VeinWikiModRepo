## Spawn System

Spawning involves:
- Spawn points in world
- Spawnlists defining what can spawn
- Probability weights
- Respawn timers
- Conditions for spawning

## Modding Guide

### Creating Custom Spawnlists

1. Define item pool
2. Set spawn probabilities
3. Configure quantities
4. Set conditions
5. Link to spawn points

### Balancing Loot

- Common items should spawn frequently
- Rare items need low probabilities
- Consider location-appropriate loot
- Balance loot with difficulty
- Test respawn rates
