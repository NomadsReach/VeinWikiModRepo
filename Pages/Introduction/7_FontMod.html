Step 7: Your Second Mod - Font Replacement

This tutorial walks you through replacing fonts used in VEIN by cooking content in your Unreal Engine project. This recreates the "Comic Sans Font Changer" Mod.

Prerequisite: You must have a properly configured Unreal Engine project from the previous step.

1. Create Folder Structure

  1. Open your "Vein" UE Project.
  2. In the content drawer, create the following folder structure: Content/Vein/UI/Fonts.

2. Import and Prepare Font

  1. Drag your desired font file (e.g., Comic Sans) into the Fonts folder in the content drawer.
  2. On the "Font Face Import Options" popup, check "Apply to All" and click "No".
  3. Copy and paste your imported font 8 times.
  4. Rename all 8 copies to these exact names, which correspond to the font assets used by the game:
    • BebasKai
    • FF_Inter
    • FF_Inter-Bold
    • FF_Inter-ExtraBold
    • FF_Inter-ExtraLight
    • FF_Inter-Light
    • Title
    • US101

3. Cook Content

  1. Click Platforms in the top menu bar.
  2. Hover over Windows, then click Cook Content.
  3. Wait for the cooking process to complete.

4. Package and Install

  1. Navigate to your cooked content folder:
    .../UnrealEngine Projects/Vein/Saved/Cooked/Windows/Vein/Content/Vein/UI/Fonts
  2. Verify that each font has .uasset, .uexp, and .ufont files.
  3. Create a new mod folder ending in _P (e.g., LxComicSans_P).
  4. Inside, create the structure: Vein/Content/Vein/UI/Fonts/
  5. Move all the cooked font files into this new folder.
  6. Package the mod using the same UnrealPak drag-and-drop method from Step 5.
  7. Install the resulting .pak file and test in-game!
